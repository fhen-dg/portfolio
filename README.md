# fhendg portfolio

**Live:** [fhendg.com](https://www.fhendg.com)

Personal portfolio built with Next.js, TypeScript, and Tailwind CSS, deployed on Vercel.

This repository showcases selected case studies and experiments across product design, system thinking, and AI-assisted development workflows.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Design workflow

Designs live in Figma. The `/design` folder holds Figma node references and visual rules captured as markdown, used as source material during implementation.

Components are built by feeding Figma designs into Cursor agents via the Figma MCP, which reads node context directly — no manual export or redline needed.

## Repository scope

This repository is intentionally focused on:

- portfolio content and case studies
- reusable UI components
- design-to-code workflow
- AI-assisted iteration
